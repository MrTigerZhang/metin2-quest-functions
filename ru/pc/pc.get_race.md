# pc.get_race()
Функция **pc.get_race** возвращает ID, уникальный для связки пол + профессия персонажа.

## Возвращаемые значения
### race
Тип *number*. ID расы персонажа.

## Примечания
Функция **не** может быть вызвана анонимно.

Существуют следующие расы:

| race | Профессия | Пол |
| --- | --- | --- |
| 0 | воин | мужчина |
| 1 | ниндзя | женщина |
| 2 | сура | мужчина |
| 3 | шаман | женщина |
| 4 | воин | женщина |
| 5 | ниндзя | мужчина |
| 6 | сура | женщина |
| 7 | шаман | мужчина |
| 8 | ликан | мужчина |
