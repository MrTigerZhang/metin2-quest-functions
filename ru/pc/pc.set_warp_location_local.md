# pc.set_warp_location_local()
Функция **pc.set_warp_location_local** устанавливает локальные координаты &laquo;выхода&raquo;, куда будет телепортирован игрок в случае вызова определенных функций

## Параметры функции
### map_index
Тип *number*. **Обязательный параметр**. Индекс локации

### local_x
Тип *number*. **Обязательный параметр**. Локальная координата по оси X без двух нулей на конце.

### local_y
Тип *number*. **Обязательный параметр**. Локальная координата по оси Y без двух нулей на конце.

## Примечания
Функция **не** может быть вызвана анонимно.
